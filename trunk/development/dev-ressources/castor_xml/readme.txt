To generate the Java classes out of the XSD files you
need
- xercesImpl-2.6.2.jar
- castor-0.9.6.jar
- log4j-1.2.9.jar
- commons-logging-1.0.4.jar

Please make sure you have downloaded and installed all these
packages (Maven/Ibiblio style).

If not, please use the provided ANT script to download all these
files.

All XSD files were developed using XMLSpy Home Edition (win32).
Please visit www.altova.com for further information.
XMLSpy Home edition is free.
